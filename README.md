# Habit Tracker App

A **web-based application** to track and manage daily habits. Built with **Python, Django, and PostgreSQL**, this app helps users create, monitor, and complete habits to improve productivity and consistency.  

---

## Features

- User registration and authentication  
- Add, view, and delete habits  
- Mark habits as completed for a day  
- Paginated habit lists for better readability  
- Responsive dashboard with dynamic updates  
- Secure: sensitive information stored in `.env`  

---

## Tech Stack

- **Backend:** Python 3.12, Django 5.x  
- **Database:** PostgreSQL  
- **Frontend:** HTML, CSS, JavaScript  
- **Libraries:** Pandas (optional CSV automation tasks)  
- **Version Control:** Git & GitHub
